# Glossary

| Term | Definition |
|---|---|
| ARMT | Agent Registry Management Tool — the system described in this SID |
| APM | Application Portfolio Management — the enterprise discipline ARMT extends into AI |
| ARB | Architecture Review Board |
| Agent | An AI-powered component capable of autonomous or assistive action |
| Assistive Agent | An agent that supports humans but does not act autonomously |
| Autonomous Agent | An agent that acts independently to complete tasks |
| Composite Agent | An agent composed of or orchestrating other agents |
| Control Point | A governance gate that must be passed before production deployment |
| Lifecycle State | The current stage of an agent: Proposed, In Review, Approved, Active, Restricted, Deprecated, Retired |
| RBAC | Role-Based Access Control |
| Risk Tier | Classification of agent risk: Low, Medium, High, Critical |
| SID | Solution Intent Document |
| TIME Model | Tolerate, Invest, Migrate, Eliminate — portfolio lifecycle framework |
